# Frontend Mentor - Tip calculator app solution

This is my solution to the Tip calculator app challenge on Frontend Mentor.

## Overview

### The challenge

Users should be able to:

- Calculate the correct tip and total cost of the bill per person
- Select a predefined tip percentage
- Enter a custom tip percentage
- See an error state if the number of people is 0
- Reset the calculator to its default state

## Built with

- HTML5
- CSS3
- Flexbox
- CSS Grid
- JavaScript

## What I learned

This project helped me improve my frontend skills, especially in:

- DOM manipulation with JavaScript
- Handling button states and user input
- Creating error states and reset functionality
- Structuring layouts with Flexbox and Grid

## AI Collaboration

I used ChatGPT during this project for:

- Debugging HTML, CSS, and JavaScript issues
- Understanding layout and DOM problems
- Implementing features like:
  - tip calculation
  - custom tip input
  - error state
  - active button state
  - reset functionality

It was especially useful for fixing structure issues and understanding why some parts of my code were not working.
